# NewBot

NewBot is a 7-DoF Feetech arm pair supported in LeRobot:

- Follower type: `newbot`, `new_bot`, or `new_arm`
- Leader type: `newbot_leader`, `new_bot_leader`, or `new_arm_leader`
- Bimanual follower type: `bi_new_arm`
- Bimanual leader type: `bi_new_arm_leader`

The default joint layout is:

1. `shoulder_roll`
2. `shoulder_pitch`
3. `elbow_roll`
4. `elbow_pitch`
5. `wrist_pitch`
6. `wrist_roll`
7. `gripper`

## Install LeRobot

Follow the main [Installation Guide](./installation), then install the hardware dependencies:

```bash
pip install -e ".[feetech]"
```

## Find the USB ports

Connect each controller board over USB and run:

```bash
lerobot-find-port
```

Use the discovered ports in the commands below. On Windows these are typically values like `COM25` and `COM26`.

## Setup motors

At the moment, `lerobot-setup-motors` does not list `newbot` in its compatible device table in this repository snapshot, so the reliable path is the Python API.

### Follower

```python
from lerobot.robots.new_arm import NewBot, NewBotConfig

config = NewBotConfig(
    port="COM25",
    id="my_newbot_follower",
)
robot = NewBot(config)
robot.setup_motors()
```

### Leader

```python
from lerobot.teleoperators.new_arm_leader import NewBotLeader, NewBotLeaderConfig

config = NewBotLeaderConfig(
    port="COM26",
    id="my_newbot_leader",
)
leader = NewBotLeader(config)
leader.setup_motors()
```

## Calibrate the arms

This repo includes dedicated calibration commands for the NewBot arm pair.

### Calibrate follower

```bash
lerobot-calibrate-new-arm-follower \
    --port=COM28 \
    --id=my_newbot_follower
```

### Calibrate leader

```bash
lerobot-calibrate-new-arm-leader \
    --port=COM26 \
    --id=my_newbot_leader
```

These commands save calibration files under your LeRobot calibration directory, for example under the Hugging Face cache.

## Teleoperate

Use the standard teleoperation CLI with the NewBot types:

```bash
lerobot-teleoperate \
    --robot.type=newbot \
    --robot.port=COM25 \
    --robot.id=my_newbot_follower \
    --teleop.type=newbot_leader \
    --teleop.port=COM26 \
    --teleop.id=my_newbot_leader \
    --display_data=true
```

If you want to attach cameras to the follower, add them with `--robot.cameras=...` the same way as the other LeRobot hardware docs.

## Bimanual NewArm

`bi_new_arm` uses one bus per arm. The right arm uses motor IDs `1` through `7`; the left arm uses motor IDs `8` through `14`. A bimanual teleoperation setup uses four ports total.

Calibrate the leader pair:

```bash
lerobot-calibrate \
    --teleop.type=bi_new_arm_leader \
    --teleop.left_arm_config.port=COM* \
    --teleop.right_arm_config.port=COM* \
    --teleop.id=my_bi_new_arm_leader
```

Calibrate the follower pair:

```bash
lerobot-calibrate \
    --robot.type=bi_new_arm \
    --robot.left_arm_config.port=COM* \
    --robot.right_arm_config.port=COM* \
    --robot.id=my_bi_new_arm_follower
```

Teleoperate with all four ports:

```bash
lerobot-teleoperate \
    --robot.type=bi_new_arm \
    --robot.left_arm_config.port=COM* \
    --robot.right_arm_config.port=COM* \
    --robot.id=my_bi_new_arm_follower \
    --teleop.type=bi_new_arm_leader \
    --teleop.left_arm_config.port=COM* \
    --teleop.right_arm_config.port=COM* \
    --teleop.id=my_bi_new_arm_leader \
    --display_data=true
```

## Python API

```python
from lerobot.robots.new_arm import NewBot, NewBotConfig
from lerobot.teleoperators.new_arm_leader import NewBotLeader, NewBotLeaderConfig

robot = NewBot(
    NewBotConfig(
        port="COM25",
        id="my_newbot_follower",
    )
)
leader = NewBotLeader(
    NewBotLeaderConfig(
        port="COM26",
        id="my_newbot_leader",
    )
)

robot.connect()
leader.connect()
```

## Current CLI support note

In this repo snapshot, `lerobot-teleoperate`, `lerobot-record`, and `lerobot-calibrate` import the bimanual NewArm modules. Use `bi_new_arm` with `bi_new_arm_leader` for paired-arm teleoperation and recording.
