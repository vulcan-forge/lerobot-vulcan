---List of commands to help train Sourccey V2 Beta

```bash
uv run -m lerobot.calibrate \
    --robot.type=sourccey_v2beta \
    --robot.id=sourccey_v2beta
```

```bash
uv run -m lerobot.calibrate \
    --teleop.type=sourccey_v2beta_teleop \
    --teleop.port=COM29 \
    --teleop.id=sourccey_v2beta_teleop
```

To teleoperate, SSH into your Raspberry Pi, navigate to the lerobot repo and run this command:

```
uv run -m lerobot.common.robots.sourccey_v2beta.sourccey_v2beta_host
```

Then on your computer run the below source command then the teleoperate command

```
source .venv/bin/activate
```

```
python examples/sourccey_v2beta/teleoperate.py
```

```
python examples/sourccey_v2beta/record.py
```

```
python examples/sourccey_v2beta/replay.py
```

```
python examples/sourccey_v2beta/evaluate.py
```

--- ACT Training

```bash
python lerobot/scripts/train.py \
  --dataset.repo_id=local/sourccey_v2beta-001__tape-a__set000 \
  --policy.type=act \
  --output_dir=outputs/train/act__sourccey_v2beta-001__tape-a__set000 \
  --job_name=act__sourccey_v2beta-001__tape-a__set000 \
  --policy.device=cuda \
  --wandb.enable=false \
  --steps=20000
```





```bash
torchrun --nproc_per_node=2 lerobot/scripts/train.py \
  --dataset.repo_id=local/sourccey_v2beta_001_tape_a  \
  --policy.type=act \
  --output_dir=outputs/train/act_sourccey_v2beta_001_tape_a \
  --job_name=act_sourccey_v2beta_001_tape_a  \
  --policy.device=cuda \
  --wandb.enable=false \
  --steps=100000 \
  --save_freq=10000 \
  --batch_size=64 \
  --distributed_training=true \
  --num_gpus=2
```
