###

Training the model

```
CUDA_VISIBLE_DEVICES=1 python src/lerobot/scripts/lerobot_train.py \
 --dataset.repo_id="Combination/sarm_2cam__sourccey-shirt-fold-01__cmbF__d__chrism__20260409__append__20260410_01" \
 --policy.type=sarm \
 --policy.annotation_mode=single_stage \
 --policy.image_keys='["observation.images.front_right","observation.images.front_left"]' \
 --output_dir=outputs/train/sarm_2cam \
 --batch_size=8 \
 --steps=20000 \
 --wandb.enable=false \
 --policy.push_to_hub=false
```
