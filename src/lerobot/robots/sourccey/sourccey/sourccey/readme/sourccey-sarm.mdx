###

Rank Episodes by Progress + Stability (No Final-Score Term)

This ranking excludes final progress from the score and prioritizes:
- high max progress
- high mean progress
- high upward ratio
- stable trajectory (low drawdown / low sudden drops)

```
python src/lerobot/robots/sourccey/sourccey/sourccey/scripts/sourccey_rank_sarm_progress.py \
 --dataset-repo-id="Combination/sourccey-shirt-fold-01__cmbF__d__chrism__20260409__append__20260410_01" \
 --progress-path="/home/sourccey/.cache/huggingface/lerobot/Combination/sourccey-shirt-fold-01__cmbF__d__chrism__20260409__append__20260410_01/sarm_progress.parquet" \
 --head-mode=sparse \
 --output-dir="outputs/sarm_episode_ranking_progress" \
 --weight-max=0.40 \
 --weight-mean=0.25 \
 --weight-upward=0.20 \
 --weight-stability=0.15 \
 --preview-top-k=25 \
 --preview-bottom-k=25 \
 --file-top-k=300 \
 --file-bottom-k=300 \
 --save-charts \
 --chart-top-k=10 \
 --chart-bottom-k=10 \
 --chart-rolling-window=61 \
 --chart-dpi=160 \
 --bottom-quality-quantile=0.10
```
